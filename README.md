# 🚀 SyncNote

SyncNote is a lightweight online workspace and file sharing platform inspired by Dontpad, designed mainly for students, labs, and quick collaboration.

Users can instantly create a workspace, write notes, upload files, and access them from any system without login.

🌐 Live Website: https://www.syncnote.in

---

# ✨ Features

- 📝 Real-time note writing
- 📂 File upload & sharing
- 📄 PDF preview support
- 🖼 Image preview support
- 🔗 Shareable workspace links
- ⚡ No signup/login required
- 📱 Mobile friendly
- 🌙 Modern UI
- ☁ Hosted on Railway
- 🌐 Custom domain support

---

# 💡 Use Cases

SyncNote is useful for:

- College labs
- Sharing program files
- Team collaboration
- Temporary workspaces
- Sharing PDFs and notes
- Accessing files from any system

# 🚀 Installation

## 1 Clone Repository

```bash
git clone https://github.com/yourusername/syncnote.git
```

## 2 Open Folder

```bash
cd syncnote
```

## 3 Install Dependencies

```bash
npm install
```

## 4 Run Server

```bash
node server.js
```
